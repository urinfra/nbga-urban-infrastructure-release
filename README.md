# NBGA Urban Infrastructure Optimization

## A nested-block scalable framework for efficient and equitable urban infrastructure management

This repository provides supporting information for the road-pipe compound-risk
optimization case reported in the associated manuscript.

Urban infrastructure systems, such as roads and pipelines, are increasingly
difficult to manage because they are large, aging, spatially distributed, and
administratively fragmented. The study develops a nested-block genetic algorithm
(NBGA) framework to support multi-year maintenance and renewal planning across
heterogeneous infrastructure assets under budgetary and temporal constraints.

The framework decomposes large-scale infrastructure networks into coordinated
nested blocks, allowing city-scale optimization problems to be represented in a
more tractable and scalable form. It is designed to evaluate investment pathways,
compound infrastructure risk, and efficiency-equity trade-offs in long-term
asset management.

### Data Use

The datasets associated with this study have been anonymized, renamed, and
reorganized. Because the original records involve urban infrastructure asset
information, the materials are provided for authorized scholarly use only.

Use of the data is limited to academic research purposes. Redistribution,
commercial use, re-identification, or use beyond the approved research scope is
not permitted.
