# Data availability

The anonymized datasets associated with the road-pipe NBGA optimization case are
provided in `data/` for authorized scholarly use only.

The repository includes only public variables prepared for academic review and
authorized research use. Original asset identifiers, source ordering, absolute
pipe calendar years, local filesystem paths, intermediate run logs, initial
asset-level risk exports, and full asset-level evolved action matrices are not
included in the public release.

When a permanent archive is created, add the repository DOI or accession
identifier here and update the manuscript Data Availability statement
accordingly.
